# speakEasy-Smart-voice-translation-assistant
## Project Description:

SpeakEasy is a web-based voice-first language assistant designed to eliminate language barriers through real-time communication. The system allows users to speak in their native language, converts the speech into text, translates it into a selected target language, and provides the translated output as voice.

This project focuses on simplicity and accessibility, making it especially useful for travelers and elderly users who may find typing difficult or complex applications hard to use. The application runs directly in a web browser and does not require installation, making it lightweight and platform-independent.

By integrating speech recognition, language translation, and text-to-speech technologies, SpeakEasy provides a seamless and efficient voice-to-voice communication experience.

## 🚀 Features
- Voice-to-voice translation
- Real-time processing
- Easy to use interface
- Web-based (no installation)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Speech Recognition API
- MyMemory Translation API
- Text-to-Speech API

## 🎯 Use Case
Helpful for travelers, elderly users, and real-time communication
